üéØ For Content Creators

‚úÖ What You Need to Know:

  • Copy and paste any YouTube, Twitter, CodePen, or GitHub Gist URL
  • Use simple shortcodes like [youtube:VIDEO_ID] for quick embedding
  • Everything is processed automatically when the post is displayed
  • No need to remember complex template syntax!

üìù In CKEditor

When using the admin interface:

  1. Just paste URLs directly into the content editor
  2. Use Templates - Click the "Templates" button for quick shortcuts
  3. Type shortcodes - [youtube:abc123] is much easier than complex template tags
  4. Preview works - What you see is what you get!
