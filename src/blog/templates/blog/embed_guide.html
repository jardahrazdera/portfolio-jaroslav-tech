  • Privacy-enhanced YouTube: Uses youtube-nocookie.com domain
  • Lazy loading: All iframes load lazily for better performance
  • Sandbox attributes: Generic embeds use iframe sandbox for security

  • Error handling: Invalid URLs show helpful error messages

Responsive Design

  • Fixed-height embeds adjust to smaller screens
  • Captions wrap gracefully on mobile devices
  • Loading states and animations for better UX

  • Reduced motion preferences respected
  • Keyboard navigation friendly
